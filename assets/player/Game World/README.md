# Player Puppet PNG Assets

Place skeletal player PNG body parts in this directory.

Expected files:
- `backpack.png`
- `head.png`
- `left_hand.png`
- `left_leg_full.png`
- `left_lower_arm.png`
- `left_thigh.png`
- `left_upper_arm.png`
- `pelvis.png`
- `right_hand.png`
- `right_leg_full.png`
- `right_lower_arm.png`
- `right_thigh.png`
- `right_upper_arm.png`
- `torso.png`

The new Flame skeletal player system loads these assets by path.
If files are missing, lightweight placeholder parts are rendered so gameplay continues.
