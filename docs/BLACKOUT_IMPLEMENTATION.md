## 🎮 Як працює Blackout Flow

```
1. Гейм-цикл → HazardSystem.update()
   ↓
2. Якщо немає активних событій, лічильник часу збільшується
   ↓
3. Після ~15 сек → _tryTriggerNextEvent()
   ↓
4. Рандомна вероятність 70% → BlackoutEvent стартує
   ↓
5. HazardSystem.isBlackoutActive = true
   ↓
6. GameHud бачить це і показує:
   - Чорний overlay з текстом "BLACKOUT"
   - Прогрес-бар (червоний) який розраховується як:
     elapsed_time / total_duration
   ↓
7. Після 3.5 сек → evento завершується
   ↓
8. isBlackoutActive = false → UI приховується
```

## 🏗️ Архітектура для майбутніх подій

Ця реалізація **повторювана** для `ToxicGasEvent` та `DoorFailureEvent`:

```dart
class ToxicGasEvent extends HazardEvent {
  ToxicGasEvent({double? duration, super.game})
    : super(
        eventId: 'TOXIC_GAS',
        duration: duration ?? GameConfig.toxicGasDuration,
      );

  @override
  void onStart() {
    // Включити постійну шкоду гравцю
    game?.applyToxicDamage();
  }

  @override
  void onEnd() {
    // Зупинити шкоду
    game?.stopToxicDamage();
  }
}
```

Просто додати у `_triggerRandomEvent()`:
```dart
else if (randomValue < 0.85) {
  // 15% для toxic gas
  _startEvent(ToxicGasEvent(game: game));
}
```
