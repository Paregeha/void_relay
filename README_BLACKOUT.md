## 🐛 Якщо щось не працює

### Затемнення не з'являється?
- Перевірте: `hazardEventsEnabled = true`
- Натиснете B для ручної активації
- Чекайте ~15 сек для авто-активації

### UI не видно?
- Переконайтесь, що немає конфліктів з іншими overlay (pause, game_over)
- Перевірте GameHud отримує `game` reference

### Прогрес-бар не змінюється?
- Перевірте `currentEventProgress` getter у HazardSystem
- Переконайтесь, що update() викликається
